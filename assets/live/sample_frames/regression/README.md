# Regression Set

Tutaj trzymaj kanoniczne klatki regresyjne live vision.

Przeznaczenie:

- klatki, ktore maja przechodzic stale po kazdej zmianie heurystyk
- sceny z ustalonym kontraktem:
  - `selected target`
  - `occupied/free`
  - minimalne zachowanie perception

To jest najlepsze miejsce na klatki podobne do:

- `live_spot_scene_1`
- `live_spot_scene_2`
- `live_spot_scene_3`
