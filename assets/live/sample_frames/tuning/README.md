# Tuning Set

Tutaj trzymaj klatki uzywane do codziennego strojenia heurystyk live vision.

Przeznaczenie:

- szybkie iteracje na `mob_a`
- szybkie iteracje na `mob_b`
- strojenie `occupied_swords`
- ROI i local confirmation

Zalecenia:

- wrzucaj tu klatki, ktore reprezentuja typowe przypadki ze spota
- do klatek dodawaj sidecary `.json`, jesli chcesz opisac:
  - `reference_point_xy`
  - `spawn_roi`
  - `expected_perception`
