# Holdout / Hard Cases

Tutaj trzymaj trudniejsze klatki, ktorych nie uzywasz do codziennego strojenia.

Przyklady:

- czesciowe zasloniecie targetu
- trudne oswietlenie
- gorszy kat kamery
- nietypowe rozmieszczenie markerow
- sceny z duzym szumem tla

Cel:

- sprawdzac, czy heurystyki nadal generalizuja
- nie stroic progow tylko pod kilka najbardziej znanych scen
